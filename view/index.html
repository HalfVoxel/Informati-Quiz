Informati-Quiz is a competitive quiz game mainly targeted to those with an interest in IT and informatics. You play against others, answering questions of varying difficulty and categories. The game is open-source and completely free to play. Questions may be proposed by adding pull requests to the games Github repository.

Currently, a Google account is necessary to play using the main server.
